### Installation

* **Install Mydia from PyPI (recommended):**

```bash
pip install Mydia
```

* **Alternatively, install from Github source:**

First, clone the repository.

```bash
git clone https://github.com/MrinalJain17/Mydia.git
```

Then, build the module

```bash
cd Mydia
python setup.py install
```
